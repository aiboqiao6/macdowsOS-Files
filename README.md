# macdowsOS Files

### Tahoe 风格的 Windows 文件浏览器

一个面向 Windows 的 Qt Quick 文件浏览器原型：用 Finder 式信息架构浏览用户目录与磁盘分区，用轻量的深色玻璃界面承载日常文件查找。

## 背景
由于我个人觉得主题修改Windows文件资源管理器仿macOS的效果过于拉跨 所以开始制作macdowsOS Files

## 介绍
macdowsOS Files 是一个使用 C++/Qt Quick 编写的第三方文件资源管理器  当前项目处于早期阶段
